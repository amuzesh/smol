Mengubah Noun menjadi Verb : Symbol  Symbolize

Pembahasan

Ada beberapa cara mengubah kata benda (Noun) ke kata kerja (Verb) dalam bahasa inggris :

1. Menambahkan akhiran –ze atau –ize pada kata benda

Contoh : symbol -> symbolize

 apology -> apologize

2. Menambahkan akhiran –en pada kata benda

Contoh : strenght -> strenghten

3. Menambahkan akhiran –fy, -ify atau –efy pada kata benda

Contoh : class -> classify

 liquid -> liquefy

4. Menambahkan akhiran –ve, untuk kata benda berakhiran –f

Contoh : belief -> believe

 proof -> prove

5. Perubahan yang tidak memiliki perubahan pola beraturan namun perubahannya sangat dekat

Contoh : sale -> sell

 seat -> sit
