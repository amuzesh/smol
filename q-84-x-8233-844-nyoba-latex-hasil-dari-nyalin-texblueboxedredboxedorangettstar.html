Jawaban: #2:

Pendahuluan :

Eksponen adalah sebuah ilmu matematika yang mempelajari sebuah hubungan antara basis dan pangkat.

Bentuk dasar dari eksponen yaitu :

  • [tex]\begingathered\texta^\textn = \underbrace\texta × a × a × ... × a_\textsebanyak n faktor\\ \endgathered[/tex]

Keterangan :

◇ a = basis

◇ n = pangkat

[tex] \: [/tex]

Eksponen mempunyai sifat sifat yaitu :

  • [tex] \sf a^m \times a^n = a^m + n [/tex]
  • [tex] \sf a^m \div a^n = a^m - n [/tex]
  • [tex] \sf \sqrt[2]a = a^ \frac12 [/tex]
  • [tex] \sf a^0 = 1[/tex]
  • [tex] \sf( a^m) ^n = a^mn [/tex]
  • [tex] \sf( \fracab ) ^n = \frac a^n b^n [/tex]
  • [tex] \sf a^ - n = \frac1 a^n [/tex]

[tex] \: [/tex]

Solusi Soal :

[tex] \sf 8^4 \times 8^233 \div 8^44 = ... \\ = \sf 8^4 + 233 \div 8^44 \\ = \sf 8^237 \div 8^44 \\ = \sf 8^237 - 44 \\ = \boxed \boxed \rm \bold193[/tex]
